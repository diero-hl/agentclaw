1. AI opens browser on VPS
2. AI goes to gmail.com and creates a new account
   - Generates a username like agentclaw.ai.XXXX@gmail.com
   - If phone verification is needed, AI uses an SMS verification service:
     - smspva.com -- buy a temp US number ($0.50), receive the code, enter it
     - sms-activate.org -- same concept, supports Gmail and X verification
   - If captcha appears, AI uses anti-captcha service (2captcha.com or anticaptcha.com)
3. AI goes to x.com and creates a new account using the Gmail it just made
   - Sets display name to "Agentclaw"
   - If phone verification is needed, uses the same SMS service
4. AI uploads profile picture and banner
5. AI writes the bio using the template
6. AI stays logged in and begins autonomous operation

### SMS Verification Services (for phone verification)
- smspva.com -- cheapest, ~$0.50 per US number
- sms-activate.org -- reliable, supports many services
- 5sim.net -- alternative option
- The AI navigates to the SMS service, buys a temp number, enters it on Gmail/X, reads the code, enters it
- All done through browser automation -- no manual steps needed

### Anti-Captcha Services (if captcha appears)
- 2captcha.com -- $3 per 1000 captchas
- anticaptcha.com -- similar pricing
- The AI sends the captcha to the service API, waits for solution, enters it

### Fallback: If Auto-Creation Fails
